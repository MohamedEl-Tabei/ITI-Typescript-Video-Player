# ITI-Typescript-Video-Player

customizable video player

## Features

- Loader
- Mute
- Play/Pause
- Time slider
- Timer
- Steps
- Theme
- npm package

## Demo

[ITI-Typescript-Video-Player Demo](iti-typescript-video-player.vercel.app)

## Installation

Install my-project with npm

```bash
  npm install mevp
```
